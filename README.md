# QuickBite üçî

QuickBite is a simple Android application built using **Java**.  
The app demonstrates basic Android concepts such as Activities, Intents, form validation, and local data storage.

## Features
- Login screen with email and password validation
- User registration with gender and date of birth
- Data passing between activities using Intent
- Local data persistence using SharedPreferences
- Logout functionality

## App Flow
Login ‚Üí Register ‚Üí Home ‚Üí Logout

## Tech Stack
- Android Studio
- Java
- XML
- SharedPreferences

## How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Wait for Gradle sync
4. Run the app on an emulator or physical device

## APK
A debug APK is included for testing purposes.

## Purpose
This project is created for learning and teaching basic Android development concepts and is suitable for beginners.
